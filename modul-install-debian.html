Instalasi Debian

Debian adalah salah satu sistem operasi berbasis Linux yang komunitasnya cukup besar. Sistem operasi Debian pertama kali dikenalkan oleh seorang mahasiswa dari Universitas Purdue, Amerika Serikat, yang bernama Ian Murdock pada 16 Agustus 1993. Berbagai aplikasi (perangkat lunak) untuk komputer bisa kamu dapatkan secara gratis pada Debian. Mulai dari aplikasi untuk keperluan mengedit teks, gambar, video, sampai ke video game. Fitur-fitur yang ada pada Debian menjadikan Debian sebagai salah satu distro terdepan saat ini. Untuk urusan server, Debian juga sudah banyak digunakan oleh perusahaan ternama.

Pada modul ini membahas bagaimana langkah untuk menginstall sistem operasi Debian. Sebelum melakukan instalasi Debian maka kalian harus terlebih dahulu menginstall Virtual Box untuk memvirtualisasikan sistem operasi Linux pada Windows atau pun Mac OS mu.

Untuk instalasi Debian sendiri terbagi menjadi dua cara yaitu instalasi berbasis GUI dan CLI. Untuk instalasi pada modul ini akan menggunakan metode instalasi Debian berbasis CLI. Berikut langkah - langkah untuk menginstall Debian pada Virtual Box.
