Konfigurasi DHCP

DHCP kepanjangan dari Dynamic Host Configuration Protocol. DHCP adalah protokol yang dipakai untuk memudahkan penyebaran alamat IP (internet protocol) secara otomatis ke perangkat lainnya.

IP Address atau alamat IP sendiri adalah barisan angka pada sebuah perangkat yang memungkinkan perangkat Anda terhubung dengan Internet. Sehingga Anda dapat mengakses halaman web, email, dan lainnya.

Berikut langkah - langkah untuk menginstall dan mengkonfigurasikan DHCP server melalui Debian di Virtual Box :

Untuk langkah awal lakukan login sebagai user pada Debian yang sebelumnya sudah diinstall sebelumnya. Seperti gambar di atas. Gunakan username dan password yang sudah dibuat sebelumnya. Untuk password memang hanya terlihat garis yang berkedip namun itu bukan masalah dan tetap ketik password yang sudah dibuat. Kemudian tekan enter.

Setelah masuk dengan akun user, maka tuliskan perintah $ su agar masuk menjadi user root dan ketik password lagi lalu tekan enter.

Selanjutnya, menginstall paket DHCP server, dengan menggunakan perintah apt install isc-dhcp-server. Setelah tampilan seperti gambar di atas tekan Y lalu enter.

Kemudian tunggu proses seperti gambar di atas. Proses di atas membutuhkan waktu beberapa menit dan kecepatan proses tersebut bergantung dengan penuh atau tidaknya storage pada komputer atau jaringan internet yang digunakan. Karena kita belum mengatur IP Address dan kebutuhan lainnya sehingga status DHCP Server masih failed.

Ketikkan perintah nano /etc/network/interfaces untuk membuat custom IP address dan subnetmask. Usahakan untuk mengikuti persis seperti yang ada pada gambar di atas guna menghindari error. Dan edit seperti di bawah.

Jika sudah diedit seperti gambar di atas maka simpan dengan tekan CTRL + X lalu tekan Y lalu tekan enter. Lalu lanjut ke tahapan selanjutnya sebagai berikut.

Perhatikan banyaknya spasi atau pun tab yang ada karena untuk menghindari error ketika menuliskan sebuah perintah karena command yang dituliskan bersifat case sensitive.

Setelah tersimpan dan tampilan seperti pada gambar di atas, ketikkan nano /etc/init.d/networking restart lalu tekan enter. Dan tunggu hingga proses selesai.

Setelah tersimpan dan tampilan seperti pada gambar di atas, ketikkan nano /etc/init.d/networking restart lalu tekan enter.

Setelah proses selesai seperti pada gambar di atas, ketikkan apt-get install isc-dhcp-server lalu tekan enter.

Kemudian ketikkan perintah seperti berikut untuk mekonfigurasi nano /etc/dhcp/dhcpd.conf lalu tekan enter.

Ketika muncul tampilan seperti pada gambar di atas, maka untuk tulisan biru berarti perintah tersebut tidak aktif sedangkan untuk mengaktifkannya hanya perlu menghapus tanda pagar "#". Untuk cara cepat menemukan perintah seperti pada gambar maka ketikkan seperti berikut CTRL + W lalu ketikkan sli lalu tekan enter dan edit IP address dan kebutuhan lainnya seperti pada gambar. Untuk menyimpannya tekan CTRL + X lalu tekan Y lalu tekan enter.

Setelah tersimpan lalu ketikkan perintah selanjutnya sebagai berikut nano /etc/default/isc-dhcp-server lalu tekan enter.

Edit konfigurasinya seperti gambar di atas yaitu menggunakan enp0s3 jika sudah lalu simpan dengan tekan CTRL + X lalu tekan Y lalu tekan enter.

Lalu restart DHCP server dengan perintah berikut nano /etc/init.d/isc-dhcp-server restart lalu tekan enter.

Tunggu proses selesai dan lihat status DHCP Server menjadi active seperti gambar di bawah.
